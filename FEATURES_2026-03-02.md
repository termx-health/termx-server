# New Features - March 2, 2026

## 📧 Email Notification System for Terminology Imports

### Quick Summary
Automated HTML email notifications sent to multiple recipients when terminology imports complete, fail, or finish with warnings.

### Supported Import Types
✅ LOINC imports  
✅ SNOMED imports (via background polling)  
✅ CodeSystem imports  
✅ ValueSet imports  
✅ MapSet/ConceptMap imports  
✅ Space imports  

### Key Features
- 🎨 Professional HTML-formatted emails
- 👥 Multi-recipient support (comma-separated list)
- 📊 Comprehensive statistics (duration, counts, messages)
- 🎯 Status-specific badges (success/warning/failure)
- 🔄 Automatic background polling for SNOMED imports
- ⚡ Asynchronous processing (doesn't block imports)
- 🛡️ Graceful degradation (works without configuration)
- 📝 Clear logging integrated with existing import logs

### Configuration
```bash
# Environment variables
SMTP_ENABLED=true
SMTP_TO_IMPORT=admin@company.org,imports@company.org
```

### Email Sample
```
Subject: [TermX] LOINC Import - Completed

✓ Completed

Job Type:     LOINC Import
Source:       LOINC
Duration:     2 min 34 sec
Successes:    1,250
Warnings:     5
Errors:       0
```

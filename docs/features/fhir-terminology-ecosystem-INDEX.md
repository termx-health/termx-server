2. Open web UI: https://dev.termx.org/tx-ecosystem/
3. Try 2-3 searches
4. Done!

### Path 2: API Developer (1-2 hours)
1. [Quick Reference](fhir-terminology-ecosystem-quick-reference.md) - Read entire document
2. [Technical Guide](fhir-terminology-ecosystem-api.md) - Read "API" section
3. Try cURL examples
4. Integrate into your code
5. Review error handling section

### Path 3: UI Power User (30 minutes)
1. [UI Guide](fhir-terminology-ecosystem-ui.md) - Read "Features" and "Usage Examples"
2. Open web UI
3. Try all filters
4. Test download functionality
5. Bookmark favorite searches

3. Set up monitoring
4. Configure custom coordination server (if needed)
5. Review performance characteristics

2. [Technical Guide](fhir-terminology-ecosystem-api.md) - Learn API details
